# Project Configuration

## Project Name
Multiplayer Grid Web App

## Project Type
Full-stack web application (React + Node.js/Express)

## Technology Stack
- **Frontend**: ReactJS with TypeScript
- **Backend**: Node.js with Express and TypeScript
- **Real-time Communication**: Socket.IO
- **Package Manager**: npm
- **Test Framework**: Playwright (E2E), Jest (Unit)

## Key Features
1. 10x10 shared grid
2. Real-time updates via Socket.IO
3. Unicode character placement
4. Player online counter
5. Cooldown mechanism (optional)
6. Historical updates tracking (optional)
